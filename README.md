# Prerequisites
* Raspberry Pi
* Raspberry Pi camera module preferably with night vision
* Motion detection software for Linux. Motion https://github.com/Motion-Project.
* Access to Azure Cognitive Services Computer Vision resource for the object(person) detection. Sign up for a free Azure account here https://azure.microsoft.com/en-gb/free/ and create a Cognitive Services resource using this guide https://docs.microsoft.com/en-us/azure/cognitive-services/cognitive-services-apis-create-account?tabs=multiservice%2Clinux
